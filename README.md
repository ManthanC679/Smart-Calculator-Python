#  Smart Calculator (Python)

##  Overview
This is a menu driven calculator built using Python.  
It allows users to perform basic arithmetic operations with input validation and stores calculation history


##  Features
- Perform addition, subtraction, multiplication, and division  
- Handles invalid inputs using exception handling  
- Prevents division by zero  
- Stores and displays calculation history  
- Menu-driven interactive interface  


##  Technologies Used
- Python  
- Functions and Loops  
- Lists (for storing history)
